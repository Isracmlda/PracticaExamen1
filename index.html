Diseño de Aplicaciones Web

Diseño de Aplicaciones Web es un curso intensivo donde aprenderás las aplicaciones y recursos para el diseño y desarrollo de sitios web que se ajusten a las demandas actuales, dentro de los prIcipales temas que conocerás están: HTML5, CSS3, JavaScript, JQuery, Responsive (Bootstrap) y Rest API. En cada lección se verá un tema específico, se darán ejemplos, prácticas y trabajos para la comprensión adecuada.

Diseño de Aplicaciones Web

El estudiante comprenderá el código HTML y su evolución a HTML5 y podrá utilizar de manera correcta las etiquetas. Entenderá la forma de aplicar e interactuar con las hojas de estilo en cascada (CSS3) de manera que permitan agilizar el proceso de diseño e implementación de un sitio web. Podrá aplicar eventos para elementos HTML con el lenguaje Javascript y podrá manipular la estructura y agregar efectos visuales con JQuery, así como consumir servicios web REST (REpresentational State Transfer) en formato de mensaje JSON y aplicación de estructura XML en los desarrollos WEB.Entre otros conceptos cubiertos está, la creación de sitios responsivos para ser visibles desde cualquier dispositivo con el uso de media queries y frameworks como Bootstrap, y por último introducirá en el desarrollo de aplicaciones del lado del servidor con el lenguaje PHP y/o ASP.net.
